# Clustertruck
<img width="1600" height="740" alt="image" src="https://github.com/user-attachments/assets/c0dace48-84c2-4d56-9fe8-6128bd4df158" />

Clustertruck is a fast-paced, first-person physics platformer developed by LANDFALL and published by tinyBuilds, where the goal is to reach the finish line by jumping on top of speeding trucks without touching the floor, which is "lava". Players navigate chaotic, increasingly difficult levels filled with hazards like lasers and hammers, using abilities like grappling hooks and jetpacks.
